# Kill-Switch Runbook

## 목적

비상 상황에서 AI 경로를 우회해 즉시 정지한다.

## 명령

- 기본 명령: `!!!!!!!`
- 스코프: 초기 운영은 `global` 고정(전체 중단)

## 절차

1. 신규 작업 접수 차단
2. 실행 중 작업 정지
3. 잠금 상태 유지
4. 사용자에게 전체 중단 상태 통지
5. 수동 해제 전까지 재개 금지
6. 호출/해제 이벤트 감사 로그 기록

## 해제 조건(최소)

1. 운영자가 중단 원인과 완화 여부를 확인
2. 수동 해제 명령(`???????`) 실행
3. 해제 직후 `status`로 핵심 상태 재확인
